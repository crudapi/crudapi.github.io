crudapi

增删改查接口

快速上手 →

无需编程

以元数据为基础，无需编程，只需配置表单和关系即可。以最少的配置帮助你专注于业务。支持主子表，级联保存等。

标准的RESTful API

默认生成CRUD API，支持标准的POST，PATCH，GET，DELETE等操作，丰富的Swagger API文档。

Web后台UI

基于Material Design风格的UI，界面美观，使用方面。支持元数据的管理和业务数据的增删改查操作！
